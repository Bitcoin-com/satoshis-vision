Email Templates

These templates were setup using Foundation for Emails! Your days of coding up painful HTML emails are officially over. .

Run 

foundation watch
in satoshis-email root. Edit files in SRC which compile down to DIST. When ready for production, run 
npm run build
to compile external stylesheets down to inline styles, making the templates work in ghetto email clients.
